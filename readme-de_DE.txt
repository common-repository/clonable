Übersetzen und pflegen Sie nahtlos Ihre mehrsprachigen Websites. Beschleunigen und vereinfachen Sie Ihre Internationalisierung mit Clonable.

== Description ==
Online-Internationalisierung ohne großen Aufwand:  Beschleunigen und vereinfachen Sie Ihre Übersetzungsprozesse. Ihre mehrsprachige Website wird automatisch aktualisiert.

= Die Herausforderung =
Die Übersetzung einer WordPress/Woocommerce-Website kann sich als kostspielig und zeitaufwändig erweisen. Neben dem anfänglichen Übersetzungsaufwand ist eine kontinuierliche Pflege unerlässlich, um sicherzustellen, dass die übersetzte Version mit neuen Inhalten aktuell bleibt. Infolgedessen werden Übersetzungen nach ihrer erstmaligen Erstellung oft übersehen oder vernachlässigt, obwohl sie auf ausländischen Märkten ein erhebliches Potenzial bieten.

= Unsere Lösung =
Wir stellen Clonable vor, eine bahnbrechende Lösung für Besitzer von WordPress-Websites. Clonable ermöglicht die mühelose Erstellung von "Klonen" Ihrer WordPress-Websites und stellt sicher, dass diese stets mit der Originalversion synchronisiert sind. Alle Änderungen, die an der ursprünglichen Website vorgenommen werden, werden sofort in den Klon übernommen, was den Wartungsaufwand erheblich reduziert. Mit Clonable können Sie einen Website-Klon in nur 5 Minuten erstellen und so die Markteinführungszeit drastisch verkürzen.

= Hauptmerkmale =
1. SEO-Verbesserung: Unser Plugin fügt nahtlos Sprach-Tags in den Kopfbereich all Ihrer Seiten ein. Dadurch wird sichergestellt, dass Ihre Klone und die ursprüngliche Website angemessen miteinander verknüpft sind, um die SEO-Leistung zu verbessern. Diese Funktionalität erstreckt sich sowohl auf übersetzte als auch auf nicht übersetzte Klone und unterstützt auch Klone von Subfoldern.
2. Mühelose Integration von Subfoldern: Mit Clonable können Sie einen Klon mühelos in einen Subfolder Ihrer Website einbinden, ohne dass komplexe technische Konfigurationen in WordPress erforderlich sind.
3. Benutzerfreundlicher Sprach-Switcher: Clonable bietet auch einen eingebauten Sprach-Switcher, mit dem die Benutzer mühelos zwischen den verschiedenen Sprachen auf Ihrer Website wechseln können.
4. Unterstützung für Woocommerce zum einfacheren tracken von Umsätzen in verschiedenen Sprachen.

== Installation ==
Sie benötigen ein [Clonable-Konto](https://app.clonable.net/register), um das Plugin nutzen zu können. Wenn Sie das Plugin erfolgreich mit Ihrem Clonable-Konto verbunden haben, werden die Einstellungen
automatisch mit Ihrer WordPress-Installation synchronisiert.

Wenn die Einstellungen nicht korrekt synchronisiert wurden, können Sie dies manuell tun, indem Sie die Schaltfläche "Mit Clonable synchronisieren" auf der allgemeinen Einstellungsseite des Plugins anklicken.
