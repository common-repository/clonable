Vertaal en onderhoud uw meertalige websites naadloos. Versnel en vereenvoudig uw internationalisatie met Clonable.

== Description ==
Online internationalisatie zonder gedoe:  Versnel en vereenvoudig uw vertaalprocessen. Uw meertalige website wordt automatisch bijgewerkt.

== De uitdaging ==
Het vertalen van een WordPress / Woocommerce website kan zowel kostbaar als tijdrovend zijn. Naast de eerste vertaalinspanning is doorlopend onderhoud essentieel om ervoor te zorgen dat de vertaalde versie actueel blijft met nieuwe inhoud. Daarom worden vertalingen vaak over het hoofd gezien of verwaarloosd na hun eerste creatie, ondanks het aanzienlijke potentieel op buitenlandse markten.

= Onze oplossing =
Maak kennis met Clonable, een baanbrekende oplossing voor WordPress website-eigenaren. Met Clonable kun je moeiteloos "klonen" van je WordPress websites maken, waarbij je er zeker van kunt zijn dat ze altijd gesynchroniseerd zijn met de originele versie. Alle wijzigingen aan de originele site worden direct doorgevoerd in de kloon, wat de onderhoudsoverhead aanzienlijk vermindert. Met Clonable kun je een website kloon genereren in slechts 5 minuten, wat de time-to-market drastisch verkort.

= Belangrijkste functies =
1. SEO Verbetering: Onze plugin voegt naadloos taal tags toe aan de head sectie van al je pagina's. Dit zorgt ervoor dat je klonen en de originele site op de juiste manier zijn gekoppeld voor verbeterde SEO prestaties. Deze functionaliteit geldt zowel voor vertaalde als niet-vertaalde klonen en ondersteunt ook klonen van submappen.
2. Moeiteloze integratie van submappen: Met Clonable kun je moeiteloos een kloon opnemen in een submap van je website, waardoor je geen complexe technische configuraties binnen WordPress nodig hebt.
3. Gebruiksvriendelijke taalswitcher: Clonable biedt ook een intuïtieve taalswitcher, waarmee gebruikers moeiteloos kunnen navigeren tussen de verschillende talen die beschikbaar zijn op je website.
4. Ondersteuning voor Woocommerce voor het makkelijker bijhouden van conversies in verschillende talen.

== Installation ==
Je hebt een [Clonable account] (https://app.clonable.net/register) nodig om gebruik te kunnen maken van de plugin. Wanneer je de plugin succesvol hebt verbonden met je Clonable account zullen de instellingen
automatisch gesynchroniseerd met je WordPress installatie.

Wanneer de instellingen niet correct gesynchroniseerd zijn, kun je dit handmatig doen door op de 'Synchroniseer met Clonable' knop te klikken op de algemene instellingen pagina van de plugin.
