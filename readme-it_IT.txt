Traducete e gestite senza problemi i vostri siti web multilingue. Velocizzate e semplificate la vostra internazionalizzazione con Clonable.

== Description ==
Internazionalizzazione online senza problemi:  Velocizzate e semplificate i processi di traduzione. Il vostro sito web multilingue si aggiorna automaticamente.

= La sfida =
La traduzione di un sito WordPress / Woocommerce può rivelarsi costosa e dispendiosa in termini di tempo. Oltre allo sforzo iniziale di traduzione, la manutenzione continua è essenziale per garantire che la versione tradotta rimanga aggiornata con i nuovi contenuti. Di conseguenza, le traduzioni vengono spesso trascurate o trascurate dopo la loro creazione iniziale, nonostante il notevole potenziale nei mercati esteri.

= La nostra soluzione =
Vi presentiamo Clonable, una soluzione innovativa per i proprietari di siti web WordPress. Clonable consente di creare senza sforzo "cloni" dei vostri siti web WordPress, assicurando che siano costantemente sincronizzati con la versione originale. Qualsiasi modifica apportata al sito originale si riflette immediatamente sul clone, riducendo in modo significativo i costi di manutenzione. Con Clonable, è possibile generare un clone del sito web in soli 5 minuti, riducendo drasticamente il time-to-market.

= Caratteristiche principali =
1. Miglioramento SEO: Il nostro plugin aggiunge senza problemi i tag della lingua alla sezione head di tutte le vostre pagine. Questo garantisce che i cloni e il sito originale siano collegati in modo appropriato per migliorare le prestazioni SEO. Questa funzionalità si estende sia ai cloni tradotti che a quelli non tradotti e supporta anche i cloni delle sottocartelle.
2. Integrazione facile delle sottocartelle: Clonable vi permette di incorporare senza sforzo un clone in una sottocartella del vostro sito web, eliminando la necessità di complesse configurazioni tecniche all'interno di WordPress.
3. Commutatore di lingua facile da usare: Clonable offre anche un commutatore di lingue intuitivo, che consente agli utenti di navigare senza sforzo tra le diverse lingue disponibili sul vostro sito web.
4. Supporto per Woocommerce per facilitare il monitoraggio delle conversioni in diverse lingue.

== Installation ==
Per utilizzare il plugin è necessario disporre di un [account Clonable] (https://app.clonable.net/register). Dopo aver collegato il plugin al proprio account Clonable, le impostazioni si sincronizzeranno automaticamente con l'installazione di WordPress.
sincronizzate automaticamente con l'installazione di WordPress.

Se le impostazioni non si sono sincronizzate correttamente, è possibile farlo manualmente premendo il pulsante "Sync with Clonable" nella pagina delle impostazioni generali del plugin.
